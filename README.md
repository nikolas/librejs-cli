librejs-cli is a command line tool for GNU LibreJS.

[![Build Status](https://travis-ci.org/nikolas/librejs-cli.svg?branch=master)](https://travis-ci.org/nikolas/librejs-cli)

Installation:

    npm install librejs-cli


Usage:

    librejs file.js
    librejs file.html
    librejs *.js


To run the tests:

    npm test
